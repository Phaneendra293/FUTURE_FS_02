# LeadFlow CRM

A client lead management system (Mini CRM) for tracking, managing, and converting incoming leads — built for agencies, freelancers, and startups.


---

## Features

- **Admin Dashboard** — Overview of total leads, new leads, conversion rate, and pipeline value with visual breakdowns
- **Lead Pipeline** — Track leads through 5 stages: New → Contacted → Qualified → Converted → Lost
- **Follow-up Notes** — Add timestamped notes to each lead for team visibility
- **Contact Form** — Public-facing form that creates leads directly into the CRM
- **Search & Filter** — Quickly find leads by name, email, or company; filter by status
- **Secure Access** — Admin-only access with email/password authentication
- **Deal Values** — Assign and edit estimated deal values per lead
- **Responsive Design** — Works on desktop, tablet, and mobile

## Tech Stack



 Frontend as React, Javascript, Tailwind CSS, HTML
Backend / Database as SQL, Node.js and Express.js
