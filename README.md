#  LeadFlow CRM

A modern **Client Lead Management System (Mini CRM)** designed for freelancers, agencies, startups, and small businesses to efficiently track, manage, and convert incoming leads.

LeadFlow helps businesses organize their sales process by managing leads through a structured pipeline, tracking follow-ups, and monitoring potential deal values—all through an intuitive and responsive interface.

#  Features

##  Admin Dashboard

* View total leads
* Monitor new leads
* Track conversion rates
* Analyze pipeline value
* Quick business insights

##  Lead Management

* Create and manage client leads
* Edit lead details
* Delete unwanted records
* Store company and contact information

##  Sales Pipeline

Track leads across five stages:

```
New
 ↓
Contacted
 ↓
Qualified
 ↓
Converted
 ↓
Lost
```

##  Follow-up Notes

* Add timestamped notes
* Maintain communication history
* Improve team collaboration

##  Contact Form

* Public-facing form
* Automatically creates new leads
* Simplifies lead collection

##  Search & Filter

* Search by:

  * New
  * Contacted
  * Qualified
  * Converted
  * Lost
##  Secure Authentication

* Admin-only access
* Protected management dashboard

##  Deal Value Tracking

* Assign estimated deal values
* Update revenue projections
* Monitor sales opportunities

##  Responsive Design

Optimized for:

* Desktop
* Tablet
* Mobile devices
